Requirement:
- For running API:
+ python3
+ flask
+ mysql
+ jwt
- For running website:
+ xampp

Using:
- Create a sql user.
- Create database funbid (empty).
- Modify your sql username and password to api.py
- Run api.py
- In funbid/api.php, configure the API_SERVER to your API server IP and port.
- Run funbid in xampp.

Images for UI using are in UI_image folder.
